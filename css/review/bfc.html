什么是 BFC？
        块级格式化上下文（Block Formatting Context，简称 BFC）是 CSS 中的一个重要概念，它是页面中的一块渲染区域，有自己的一套渲染规则，决定了其子元素将如何定位，以及和其他元素的关系和相互作用。

        BFC 可以被看作是页面中的一个独立的容器，容器内部的元素不会影响到外部元素，外部元素也不会影响到容器内部的元素。

        BFC 的特性
        BFC 具有以下关键特性：

        包含内部浮动：BFC 会包含内部的浮动元素，防止高度塌陷
        排除外部浮动：BFC 不会与浮动元素重叠

        阻止外边距重叠：BFC 可以阻止元素的外边距（margin）与其他元素的外边距发生重叠
        BFC阻止外边距重叠的原理：
        BFC会阻止内部元素与外部元素的外边距重叠
        但不会阻止BFC元素本身与相邻兄弟元素的外边距重叠

        Q:为什么display:inline-block能阻止外边距重叠?
        A:inline-block 使元素不再是纯块级元素.纯块级元素才会发生相邻兄弟元素的上下外边距重叠
          行内块级元素（inline-block）、浮动元素、绝对定位元素等不会与相邻元素发生上下外边距重叠

        独立的容器：BFC 内部的元素不会影响外部元素的布局
        每个元素的左外边缘接触包含块的左边缘：对于从左往右的格式化，反之亦然
